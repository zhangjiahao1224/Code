# LeetCode Practice Folder

This folder is for practicing LeetCode problems using Python.

## How to Use
- Create a subfolder for each problem or use naming conventions like `problem_XX.py`.
- Write your solution and test cases in each file.
- You can also use this folder to track your progress.

## Tips
- Focus on understanding the problem before coding.
- Write clean, readable code with comments.
- Test your solution with edge cases.
- Consider time and space complexity.

Happy coding!
