# Deep Learning 课程实践总结

本目录包含深度学习基础与序列建模的练习代码，目标是通过代码实现理解从基础前馈网络到现代 Transformer 架构的演进逻辑。

## 目录结构

- `Class_1.py`：前馈神经网络（MLP）、二分类任务训练流程与正则化技术。
- `Class_2.py`：序列建模进阶，涵盖 RNN 基础、自注意力（Self-Attention）及 Transformer Block 实现。

## 第 1 课：前馈神经网络与训练正则化

本节重点在于跑通监督学习的完整工程链路，理解如何构建稳定且具备泛化能力的模型。

### 核心目标

- 搭建基础二分类神经网络。
- 掌握训练集与验证集划分、损失函数选择及优化器配置。
- 通过正则化手段对抗过拟合，提升模型泛化性能。
- 学会模型保存、早停和训练可视化。

### 关键技术

| 概念 | 说明 |
| :--- | :--- |
| `MLP` | 多层感知机，深度学习中的基础前馈网络。 |
| `BCEWithLogitsLoss` | 二分类常用损失函数，将 Sigmoid 与交叉熵合并，数值更稳定。 |
| `Dropout` | 训练时随机失活部分神经元，降低过拟合风险。 |
| `Weight Decay` | 对权重增加惩罚项，限制参数过大。 |
| `AdamW` | 更适合配合权重衰减的优化器。 |
| `Early Stopping` | 监控验证集表现，在不再提升时提前停止训练。 |
| `Checkpoint` | 保存模型权重、结构配置和训练历史。 |

### 核心逻辑流

```text
数据准备 -> 数据增强 -> 模型定义 -> 划分数据集 -> 训练循环 -> 验证评估 -> 保存最佳模型 -> 可视化
```

### 这一课最该记住的内容

- 模型不仅要学会训练集，还要具备泛化能力。
- 正则化的本质是约束模型复杂度，降低过拟合风险。
- 一个完整训练脚本通常不仅有模型，还包括数据处理、验证评估、模型保存和可视化。

## 第 2 课：序列建模，从 RNN 到 Transformer

本节探讨当数据具备时序性和上下文依赖时，模型架构应如何演进。

### 序列建模的核心问题

序列任务与普通独立样本的区别在于：

- 输入有顺序。
- 当前输出往往依赖过去信息。
- 样本长度可能不同。
- 重要线索可能出现在很久之前。

典型任务包括：

- 文本分类
- 语言建模
- 机器翻译
- 图像描述
- 音乐生成

### RNN 的核心思想

RNN 通过隐藏状态在时间上递推：

```text
h_t = f(x_t, h_{t-1})
```

其中：

- `x_t`：当前时间步输入。
- `h_{t-1}`：上一时刻隐藏状态。
- `h_t`：当前新的隐藏状态。

这意味着模型会把当前输入和过去记忆结合起来，形成新的内部状态。

### RNN 的优点与局限

优点：

- 可以处理变长序列。
- 天然保留顺序信息。
- 参数在各时间步共享。
- 结构直观，适合入门理解序列模型。

局限：

- 长期依赖难学。
- 容易出现梯度消失或梯度爆炸。
- 必须按时间步顺序计算，并行效率较低。
- 长序列任务上通常不如更新的架构。

### 为什么需要 LSTM / GRU

普通 RNN 对长期记忆的保持能力有限，因此发展出了门控循环结构：

- `LSTM`
- `GRU`

它们的核心思想是让模型学会选择性记忆和遗忘，也就是通过门控机制控制：

- 哪些旧信息保留
- 哪些新信息写入
- 哪些信息输出

### 为什么发展到 Attention

RNN 的问题是远处信息必须一步一步往后传，路径太长。Attention 的思想是：

```text
当前预测时，直接去看最相关的位置
```

这样模型就不必完全依赖隐藏状态跨很多步传递信息。

### Self-Attention 的核心公式

Self-Attention 从输入中提取三组表示：

- `Q`：Query
- `K`：Key
- `V`：Value

核心公式：

```text
Attention(Q, K, V) = softmax(QK^T / sqrt(d_k)) V
```

直观理解：

- `QK^T`：计算谁和谁更相关。
- `softmax`：把相关性变成注意力权重。
- `乘 V`：把重要位置的信息加权汇总。

### Transformer 的核心组成

Transformer 不再依赖 RNN 式递推，而是主要依赖：

- `Embedding`
- `Position Embedding / Positional Encoding`
- `Self-Attention`
- `Feed Forward Network`
- `Residual + LayerNorm`

最重要的区别是：

- `RNN`：按顺序读，靠隐藏状态记忆。
- `Transformer`：直接看全局，决定该关注谁。

### `Class_2.py` 在做什么

当前代码是一个教学版最小示例，主要包含：

- 字符级词表构建
- 文本编码与解码
- 语言模型训练样本构造
- 简单的 `RNN` 语言模型
- 简单的 `SelfAttention`
- 极简 `Transformer Block`
- 极简 `Transformer` 语言模型
- 自回归文本生成
- `Q / K / V` 维度演示

### 这一课最该记住的内容

- 序列建模的关键是顺序和上下文。
- `RNN` 是经典起点，但有长期依赖和并行效率问题。
- `LSTM / GRU` 用门控机制改进记忆能力。
- `Self-Attention` 允许模型直接关注远距离信息。
- `Transformer` 是现代大语言模型的基础结构。

## 两节课之间的演进逻辑

可以把两节课的主线概括为：

```text
基础神经网络训练
-> 前馈模型与正则化
-> 序列建模需求
-> RNN
-> 长期依赖问题
-> Attention
-> Transformer
```

它们分别回答了两个问题：

- 第 1 课：神经网络怎样训练得更稳、更好。
- 第 2 课：当数据有顺序关系时，模型怎样理解上下文。

## 复习建议

1. 先看 `Class_1.py`，重点理解训练循环、损失函数、优化器和正则化。
2. 再看 `Class_2.py`，重点理解 RNN 和 Transformer 的建模差异。
3. 把 `h_t = f(x_t, h_{t-1})` 和 `Attention(Q, K, V)` 两个公式吃透。
4. 复习时优先抓主线：为什么会从前馈网络发展到序列模型，再发展到 Transformer。

## 一句话总结

第 1 课教的是如何科学地训练模型，第 2 课教的是如何优雅地建模序列，它们共同构成了从基础深度学习训练到现代序列模型的入门框架。
