# 屏幕识别贪吃蛇机器人

说明：这是一个通过屏幕识别并自动按键来玩的简单贪吃蛇机器人。它通过把截图区域划分为固定网格，基于像素颜色判断蛇身与食物，并用 BFS 寻路到食物，然后发送方向键。

使用前准备:
- 在 Windows 上安装 Python 3.8+。
- 建议在虚拟环境中安装依赖：

```bash
python -m pip install -r snake_bot/requirements.txt
```

运行方法:

```bash
python snake_bot/bot.py --grid-w 20 --grid-h 20
```

参数说明：
- `--grid-w` `--grid-h`：指定网格列数与行数（根据游戏分辨率与方块大小调整）。
- 运行后会提示你选择游戏区域的左上角和右下角（把鼠标移动到位置并按回车）。

注意事项与调优：
- 颜色阈值是基于常见绿色蛇身和红色食物的启发式判断，可能需要根据具体网页或客户端游戏调整阈值或改用模板匹配。
- 请在安全且允许自动按键的环境下使用。按 Ctrl+C 可退出程序。
