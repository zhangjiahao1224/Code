# Python 学习路径

本目录存放 Python 学习过程中的笔记与示例代码。

## 章节说明

| 章节 | 文件夹 | 主要内容 |
|------|--------|----------|
| **1** | [01-Basics](./01-Basics/) | 变量、数据类型、运算符、流程控制（if/else、for、while） |
| **2** | [02-Functions](./02-Functions/) | 函数定义、参数、返回值、递归、lambda表达式 |
| **3** | [03-Data-Structures](./03-Data-Structures/) | 列表、元组、字典、集合、字符串操作 |
| **4** | [04-OOP](./04-OOP/) | 类、封装、继承、多态、特殊方法 |
| **5** | [05-File-Handling](./05-File-Handling/) | 文件读写、CSV处理、JSON处理、路径操作 |
| **6** | [06-Modules-and-Packages](./06-Modules-and-Packages/) | 模块导入、包结构、标准库概览、虚拟环境 |
| **7** | [07-Exception-Handling](./07-Exception-Handling/) | 异常处理、自定义异常、断言、日志记录 |
| **8** | [08-Standard-Library](./08-Standard-Library/) | 常用标准库：os, sys, datetime, random, math, itertools |

## 建议学习顺序

按上述 1～8 章顺序依次学习，后一章会用到前一章的概念。
