# C++ 学习路径

本目录存放 C++ 学习过程中的笔记与示例代码。

## 章节说明

| 章节 | 文件夹 | 主要内容 |
|------|--------|----------|
| **1** | [01-Basics](./01-Basics/) | 变量、数据类型、运算符、流程控制（if/else、for、while） |
| **2** | [02-Functions](./02-Functions/) | 函数定义、参数、返回值、重载 |
| **3** | [03-Arrays-and-Pointers](./03-Arrays-and-Pointers/) | 数组、指针、引用 |
| **4** | [04-OOP](./04-OOP/) | 类、封装、继承、多态 |
| **5** | [05-STL](./05-STL/) | 标准库：容器、算法、迭代器 |
| **6** | [06-Memory-Management](./06-Memory-Management/) | `new`/`delete`、RAII、智能指针 |
| **7** | [07-Advanced-Topics](./07-Advanced-Topics/) | 模板、异常、多线程、现代 C++ 特性 |

## 建议学习顺序

按上述 1～7 章顺序依次学习，后一章会用到前一章的概念。
